# Argya Sahaya


Arogya Sahaya is a healthcare assistance mobile application designed specifically for **rural elderly people in Karnataka**, aiming to improve healthcare accessibility, medication adherence, emergency response, and health monitoring through a simple and user-friendly Android application.

## Problem Statement
Rural elderly populations often face challenges such as:

- Difficulty remembering medication schedules
- Limited access to healthcare information
- Lack of quick emergency communication
- Poor tracking of health vitals
- Difficulty discovering nearby health camps and services

Arogya Sahaya addresses these issues through a digital healthcare companion built with accessibility in mind.

---

## Features

###  Authentication
- User login and signup using Firebase Authentication
- Secure user access

###  Profile Management
- Store personal health information


###  Medicine Reminder System
- Add medicine schedules
- Track dosage and timings
- Mark medicines as:
  - Taken
  - Pending
- Local notification reminders

###  Vital Health Monitoring
Track important health vitals:

- Blood Pressure
- Blood Sugar
- Heart Rate
- Date-wise logging

###  SOS Emergency Alert
Emergency support feature that:

- Retrieves emergency contact from profile
- Sends emergency SMS alert
- Fast access during emergencies
